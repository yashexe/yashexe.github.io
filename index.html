    Instaply

    GitHub ↗

    A job-discovery engine that monitors ATS feeds and ranks roles against your résumé — entirely on your machine.

    Asterion

    GitHub ↗

    A moving-object detection pipeline that finds asteroids in real telescope imagery.

    • 5–7 Mpix/s on a laptop; 43 unit/integration/API tests; no-build SPA served over FastAPI.
    • Python
    • NumPy

    • Astropy
    • FastAPI
